# github-user-lookup

**Task:** github-user-lookup  
**Round:** 1

## Summary
This repository contains a minimal, production-ready single-page application generated by an LLM (Gemini).  
Task brief: Create a web page that has a form with an input id="github-user-${seed}". On submit, show that GitHub user's account creation date (YYYY-MM-DD in UTC) within 5 seconds. If the request contains a ?token=... then use that as the GitHub API token.

## Setup
1. Clone the repository:
```bash
git clone https://github.com/Tusharisme/github-user-lookup.git
cd github-user-lookup
```

Files are static — open index.html or deploy with GitHub Pages.

## Usage

Open index.html in a browser or visit the GitHub Pages URL (if enabled).
If the task supplied attachments (CSV, images), they are expected in the /input folder at the repo root when deployed.

## Code Explanation

index.html — single-page app (HTML/CSS/JS). The app loads attachments from input/ (if present).

The app is intentionally lightweight, uses vanilla JS and minimal CDN libraries where necessary (e.g., PapaParse, Chart.js) to remain client-side only.

## Deployment

This repo is configured to be served via GitHub Pages (branch main, root directory).
